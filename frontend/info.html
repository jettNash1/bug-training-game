How Quizzes Work

Question Format

Each quiz has exactly 15 questions
Each question has one correct answer
Questions progress through 3 difficulty levels

Scoring System

Score calculated as percentage of correct answers
80% or higher is considered passing
You can retake quizzes to improve your score

Progress Saving

Progress is automatically saved after each question
You can resume where you left off
No need to complete in one session

Timer System

Understanding how the quiz timer works:

Default Timer

Default time limit set by administrators
Some quizzes may have timers disabled
Timer shows in top-right during questions

Timer Persistence

Timer state is saved if you leave mid-question
Timer resumes from where you left off
Works across browser tabs and sessions

Time Expiry

When timer reaches 0, question is marked incorrect
Random answer selected if none chosen
Quiz automatically proceeds to next question

Badges System

Earn badges to showcase your achievements:

How to Earn Badges

Complete quizzes with 80% or higher score
Each quiz has its own unique badge
Badges are awarded immediately upon completion

Badge Collection

View all your badges on the Badges page
Badges show the date earned
Share your achievements with others

Retaking Quizzes

You can retake quizzes to improve scores
Higher scores replace previous attempts
Badge eligibility updates with new scores

Need Help?

Quiz Reset Requests

If you've failed a quiz and want to start over completely, contact an administrator to request a quiz reset. This will:

  • Clear all your progress for that specific quiz
  • Remove any failing score
  • Allow you to start fresh with a clean slate

Technical Support

For technical issues, questions about the system, or quiz content feedback:

  • Contact your system administrator
  • Provide details about the specific quiz or issue
  • Include any error messages you encountered

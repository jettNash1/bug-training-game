User Management

Overview

The Users section provides comprehensive tools for managing learners and tracking their progress across all quizzes.

Key Features

  • Statistics Dashboard: View total users, daily active users, and overall progress metrics
  • User Search & Filtering: Find users by username, sort by progress or activity
  • View Options: Switch between grid and row views for user display
  • User Details: Click any user to see detailed quiz progress, scores, and activity history
  • Progress Management: Reset individual quiz progress or all progress for a user
  • Account Deletion: Remove user accounts when necessary

User Card Information

  • Progress Bar: Shows percentage of quizzes completed
  • Last Active: When the user last interacted with the system
  • Quiz Status Colors: White (not started), light cream (in progress), light orange (completed but below 80%), light green (80%+ pass)

Settings

Quiz Timer Settings

Configure time limits for quiz questions to ensure consistent testing conditions.

  • Global Default: Set the default timer for all quizzes
  • Per-Quiz Timers: Override the default for specific quizzes
  • Timer Persistence: Timers are saved if users leave mid-question
  • Auto-Fail: Questions are marked incorrect when time expires

Guide Settings

Manage external guide links that appear as buttons on quiz pages.

  • Enable/Disable: Turn guide buttons on or off for each quiz
  • URL Configuration: Set the destination URL for guide buttons
  • Real-time Updates: Changes appear immediately for users
  • Bulk Operations: Enable/disable multiple guides at once

Auto-Reset Settings

Configure automatic quiz resets that run at regular intervals.

  • Reset Periods: Set daily, weekly, or custom intervals
  • Quiz-Specific: Different reset schedules for different quizzes
  • Countdown Displays: See time remaining until next reset
  • Manual Triggers: Force immediate resets when needed

Create Account

Account Creation

Create new user accounts with customized quiz visibility.

  • Username & Password: Set secure login credentials
  • Quiz Visibility: Select which quizzes the user can access
  • Category-Based Selection: Choose quizzes by functional area
  • Bulk Selection: Enable/disable entire categories at once
  • Custom Access: Fine-tune access for specific learning paths

Quiz Categories

  • Core Testing: Fundamental QA skills and mindset
  • Test Types: Different testing approaches and methodologies
  • Process & Management: Test planning, reporting, and team coordination
  • Tools & Technology: Automation, CMS testing, and technical skills
  • Specialized Testing: Mobile, performance, security, and accessibility

Schedule

Manual Scheduled Resets

Schedule one-time quiz resets for specific users at precise dates and times.

  • User Selection: Choose from all registered users
  • Quiz Selection: Pick specific quizzes to reset
  • Date & Time: Set exact reset timing
  • Timezone Handling: Automatic conversion to user's local time
  • Schedule Management: View, edit, or cancel scheduled resets

Use Cases

  • Retesting: Allow users to retake failed assessments
  • Training Cycles: Reset progress for new training periods
  • Issue Resolution: Clear problematic progress states
  • Batch Operations: Coordinate resets across multiple users

Quiz Questions

Question Management

View and manage the question bank for all quizzes in the system.

  • Question Browser: Browse questions by quiz and difficulty level
  • Content Review: View question text, answers, and explanations
  • Difficulty Levels: Questions are organized into three tiers
  • Answer Key: See correct answers and explanations
  • Usage Statistics: Track which questions are being used

Question Organization

  • By Quiz: Questions grouped by subject area
  • By Difficulty: Progressive complexity from basic to advanced
  • Randomization: Questions are selected randomly during quizzes
  • Pool Management: Ensure sufficient questions for fair testing

Export

Data Export Options

Export user progress and system data for analysis and reporting.

Export Types

  • All User Data: Complete export of all users and their progress
  • Simple CSV: Basic user information and completion status
  • Custom Export: Select specific users and quizzes for targeted reports

Custom Export Features

  • User Selection: Choose individual users or groups
  • Quiz Selection: Filter by specific quizzes or categories
  • Data Fields: Control which information is included
  • Format Options: CSV format for spreadsheet compatibility
  • Progress Tracking: Monitor export generation progress

Use Cases

  • Performance Reports: Analyze learning outcomes
  • Compliance Documentation: Track training completion
  • Data Analysis: Identify learning patterns and trends
  • Backup: Create data backups for security

User Badges

Badge Management

View and manage user achievements and badge awards.

  • User Badge Views: See all badges earned by each user
  • Badge Details: View achievement dates and criteria
  • Progress Tracking: Monitor which users are close to earning badges
  • Achievement Analytics: Understand learning progression

Badge System

  • Automatic Awards: Badges earned automatically upon achieving 80%+ score
  • Quiz-Specific: Each quiz has its own unique badge
  • Retake Support: Higher scores update badge eligibility
  • Visual Recognition: Unique badge designs for different achievements

Best Practices

User Management

  • Regular Monitoring: Check user progress and activity regularly
  • Progressive Access: Consider unlocking quizzes gradually based on progress
  • Support Responsiveness: Address reset requests and technical issues promptly
  • Data Privacy: Handle user data according to your organization's privacy policies

System Maintenance

  • Timer Settings: Adjust timers based on question complexity and user feedback
  • Guide Links: Keep external guide URLs current and accessible
  • Auto-Reset Schedules: Align reset periods with training schedules
  • Regular Exports: Create periodic data backups

Performance Monitoring

  • Usage Analytics: Track which quizzes are most/least used
  • Completion Rates: Monitor overall system engagement
  • Issue Tracking: Identify and resolve common user problems
  • Feedback Integration: Use export data to improve learning content
